# UI/UX 全面重新設計

## 設計原則

1. **快** — 能用 postback/clipboard 就不開 LIFF
2. **美** — 極簡、留白、資訊分層、進度條視覺化
3. **簡** — 每個操作最多 2 步完成

## 架構變更

### 需要 LIFF 的操作（2 個）

- **開團** — 表單輸入多，必須用網頁（欄位不變，只改外觀）
- **找團** — 多重篩選 + 結果列表

### 改用 Flex + postback 的操作

- **我的團** — Bot 推送 Flex 卡片列表 + postback 按鈕
- **訂閱設定** — Bot 推送 Flex 選單 + postback + 文字輸入
- **加入團** — 卡片上 postback 直接加入（不開 LIFF）
- **分享** — clipboard action（社群）+ shareTargetPicker（好友/群組）

### Rich Menu（2x2）

```
┌────────────┬────────────┐
│   🚪 開團   │   🔍 找團  │  ← 開 LIFF
├────────────┼────────────┤
│   📋 我的團 │   🔔 訂閱  │  ← postback，Bot 回覆 Flex
└────────────┴────────────┘
```

## Flex 卡片設計

### 團卡片（分享出去的版本）

白底、無 header 色塊、左對齊、資訊壓縮。

```
┌─────────────────────────┐
│  🎯 密室逃脫大冒險       │  房間名（bold lg）
│  笨蛋工作室              │  工作室（gray xs）
│                         │
│  📅 4/5(六) 14:00       │  日期時間一行
│  📍 台北 · 90分 · $500   │  地點/時長/費用一行
│                         │
│  ■■■□□□  3/6 人         │  進度條 + 人數
│  還差 3 人 · 團主：Alice  │
│                         │
│  ┌─────────────────┐    │
│  │    我要加入 ✋    │    │  postback action
│  └─────────────────┘    │
└─────────────────────────┘
```

進度條實作：用 `■` 和 `□` 或用多個小 box 元素拼接。

### 團主版卡片（開團成功後）

```
┌─────────────────────────┐
│  ✅ 開團成功              │
│                         │
│  [團資訊同上]             │
│                         │
│  [分享卡片 📤]           │  shareTargetPicker
```

標題 bubble + 各團 bubble（kilo size），保持 carousel 格式但套用新卡片風格。

## 開團後完整流程

```
開團
  → 團主收到團主版卡片（分享/複製按鈕）
  → 訂閱者收到團卡片 + 加入按鈕
  → 管理員收到團卡片

有人加入
  → 團主收到加入通知小卡
  → 加入者收到確認文字

成團（滿員）
  → 團主收到成團通知 + [建立群組] 按鈕
  → 所有成員收到成團通知

團主建群
  → 團主貼群組邀請連結
  → Bot 轉發連結給所有成員

活動前 24 小時
  → 所有成員 + 團主收到提醒

退團
  → 發送申請給團主
  → 團主同意/拒絕 (postback)
  → 通知雙方結果

取消團
  → 通知所有成員「團已取消」
```

## 我的團（Flex + postback）

點 Rich Menu「我的團」→ Bot 回覆：

### 有團時

Carousel，每個團一個 bubble：

```
┌──────────────────────┐
│ 密室大冒險      [募集中] │
│ 4/5(六) 14:00 · 3/6人 │
│ ■■■□□□               │
│                      │
│ [管理成員] [取消團]    │  postback
└──────────────────────┘
```

管理成員 → Bot 回覆成員列表 Flex（每人一行 + 踢出按鈕）

### 沒有團時

```
還沒有開過團，點選「開團」建立第一個團！
```

## 訂閱設定（Flex + postback）

點 Rich Menu「訂閱」→ Bot 回覆：

```
┌──────────────────────┐
│ 🔔 訂閱設定            │
│                      │
│ 目前訂閱：             │
│  📍 台北  [✕]         │
│  🔍 密室逃脫  [✕]     │
│  💰 500元以下  [✕]    │
│                      │
│ [+ 地區] [+ 關鍵字]   │
│ [+ 價格] [+ 星期]     │  postback
└──────────────────────┘
```

點 [+ 地區] → Bot 回覆地區選擇 carousel（北中南東各一 bubble）
點 [+ 關鍵字] → Bot 回覆「請輸入關鍵字」
點 [+ 價格] → Bot 回覆「請輸入價格上限」
點 [+ 星期] → Bot 回覆星期選擇 Flex

## LIFF 頁面（只剩 2 個）

### 開團表單

- 欄位不變
- 外觀翻新：更大的字體、更多留白、更好的間距
- 載入優化：skeleton loading
- 成功後直接顯示團主版卡片按鈕（不跳轉）

### 找團

- 篩選區更直覺
- 結果用新卡片風格
- 加入按鈕改 postback（不在 LIFF 裡 fetch API）
  → 點加入 → 呼叫 API → 顯示結果 toast
  （LIFF 內沒有 postback，所以找團頁面的加入仍用 fetch API）

## 實作優先順序

1. **團卡片重新設計** — 影響最大，所有分享/通知都用到
2. **加入改 postback** — 不開 LIFF，體驗大幅提升
3. **團主版卡片 + clipboard** — 分享流程簡化
4. **通知訊息重新設計** — 加入/成團/退團/提醒
5. **我的團改 postback** — 移除 LIFF
6. **訂閱改 postback** — 移除 LIFF
7. **Rich Menu 更新** — 配合新流程
8. **開團 LIFF 翻新** — 外觀改善
9. **找團 LIFF 翻新** — 外觀改善
10. **每日彙整 + 全部團複製** — 新功能
